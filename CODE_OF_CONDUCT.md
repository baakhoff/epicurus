# Code of Conduct

epicurus adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. Please read it — by participating in this project (issues, pull
requests, discussions) you agree to uphold it.

In short: we're committed to a respectful, welcoming, and harassment-free community for
everyone, regardless of background or identity.

## Reporting

Raise concerns privately with the maintainer at **a@baakhoff.com**. Reports are handled
confidentially. The maintainer is responsible for enforcement and may remove, edit, or
reject contributions and comments that don't align with this Code of Conduct.

The full standards, enforcement responsibilities, and guidelines are in the Contributor
Covenant v2.1 linked above.
